Sobre as aulas programação

O objetivo desta página é apresentar aos estudantes a Programação como prática pedagógica para o desenvolvimento do pensamento computacional, uma vez que o ato de programar envolve criar soluções para a resolução de problemas diversos, utilizando habilidades como abstração e análise.

A presente Trilha de Aprendizagem abordará a Linguagem de Marcação de Hipertexto - HTML, responsável por definir o conteúdo da página, para que esse seja exibido em um navegador da web; a Linguagem de Folha de Estilo em Cascata - CSS, cuja aplicação relaciona-se com a definição de estilos que controlam a aparência da página e a Linguagem de Programação JavaScript, para tornar a página mais dinâmica, adicionando a ela elementos interativos.

Com o intuito de aplicar os conhecimentos e desenvolver as habilidades previstas para o componente, ao longo do ano, o estudante desenvolverá um projeto prático, que consiste na criação de uma página web responsiva, utilizando HTML, CSS e JavaScript, sendo ela o produto pedagógico esperado.
